Реализовать функционал компонента «Пожаловаться на новость», который позволит собирать мнение посетителей сайта о размещенных новостях.

В визуальном редакторе компонент находится в разделе Контент - Пожаловаться на новость.

Для лучшей работы компонента, его вызов лучше всего расположить в файле component_epilog.php, в шаблоне компонента news.detail.

Пример вызова компонента:
<?$APPLICATION->IncludeComponent(
	"custom:report.news",
	"",
	Array(
		"AJAX_MODE" => "Y",
		"IBLOCK_ID" => "3",
		"IBLOCK_TYPE" => "news",
		"NEWS_ID" => $arResult["ID"]
	)
);?>
